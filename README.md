开发这个扩展的原因是业务上经常会有把数据生成特定xlsx格式导出,根据日常开发的一些需求特点,集成一些功能,争取通过最少的配置来实现需求,而不需要去看对应的  PhpSpreadsheet 包文档

### 环境

```php
php >7.1
Laravel 版本 6.*
```

## 安装(如碰到缺少扩展的,请对应先安装扩展)

```bash
composer require remember/laravel-spreadsheet
```
